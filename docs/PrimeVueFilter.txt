PrimeVue DataTable: фильтры (заметки по проекту)

- Базовый компонент: `resources/ts/components/BaseDataTable.vue` (filterDisplay="row").
- Маппинг фильтров в DTO: `resources/ts/utils/filterMapper.ts`.
  - text/select -> filter[field] = value
  - number -> Number(value)
  - boolean -> 1/0
  - dateRange -> { from, to } из диапазона дат

Практика
- Для range-фильтра используем Calendar с `selectionMode="range"` (value = [from, to]).
- Для select/boolean фильтров передаем options из Pinia dictionaries.
- Prefer field names с `cashbox_id`. Бэкенд принимает `cash_box_id`, но это legacy.

Важно
- Если используешь `matchMode="custom"`, обязательно задай `filterFunction` в Column.
  Иначе PrimeVue кидает ошибку `filterConstraint2 is not a function`.
- В PrimeVue v4 Calendar помечен к замене на DatePicker. В проекте пока оставлен Calendar.
